# 🛒 Amazon Clone  

A full-stack **Amazon-inspired e-commerce web application** built to demonstrate modern web development skills, including responsive UI, authentication, product management, and cart functionality.  

---

## 📌 Project Overview  
This project replicates core features of Amazon, showcasing practical implementation of:  
- **User Authentication** (Sign up, Login, Logout)  
- **Product Listings** with dynamic rendering  
- **Shopping Cart** functionality  
- **Order Management**  
- **Responsive Design** for seamless experience across devices  

The goal is to highlight **hands-on coding ability** and **real-world problem-solving skills** in web development.  

---

## 🚀 Tech Stack  
| Layer          | Technologies Used                                  |
|----------------|----------------------------------------------------|
| Frontend       | HTML, CSS, JavaScript                              |
| Styling        | Bootstrap                                          |
| Backend        | Node.js, Express.js                                |
| Database       | MongoDB                                            |
| Authentication | Firebase/Auth or JWT (depending on implementation) |

---

## 🧑‍💻 Features  
- 🔐 **Secure Authentication** – Users can register and log in.  
- 🛍️ **Product Catalog** – Browse products with clean UI.  
- 🛒 **Cart System** – Add/remove items, update quantities.  
- 📦 **Order Placement** – Simulated checkout flow.  
- 📱 **Responsive Layout** – Optimized for mobile and desktop.

## 🛠️ Installation & Setup  
1. Clone the repository:  
   ```bash
   git clone https://github.com/ronitgupta138/Amazon-Clone.git
   ```
2. Navigate to the project folder:  
   ```bash
   cd Amazon-Clone
   ```
3. Install dependencies:  
   ```bash
   npm install
   ```
4. Start the development server:  
   ```bash
   npm start
   ```
5. Open in browser:  
   ```
   http://localhost:3000
   ```
